**1. Create `backend/app/routers/answers.py`:**

Module-level docstring. Create `router = APIRouter(prefix="/api/answers", tags=["answers"])`.

**POST `/submit` endpoint** (response_model=AnswerSubmitResponse):
- Accepts `AnswerSubmitRequest` body and `db: Session = Depends(get_db)`
- Queries `Question` by `request.question_id` with eager-loaded `answer_options`
- Returns 404 if question not found
- Finds the correct AnswerOption (where `is_correct == True`)
- Finds the user's selected AnswerOption (where `label == request.selected_option_label`)
- Returns 400 if selected_option_label doesn't match any option
- Builds `OptionFeedback` list from all answer_options
- Returns `AnswerSubmitResponse` with:
  - `is_correct = (request.selected_option_label == correct_option.label)`
  - `correct_option_label = correct_option.label`
  - `correct_option_text = correct_option.text`
  - `explanation = question.explanation`
  - `options_feedback = [OptionFeedback(label=o.label, text=o.text, is_correct=o.is_correct, explanation=o.explanation) for o in question.answer_options]`

**2. Add POST `/explain` to `backend/app/routers/claude.py`:**

Add a new endpoint below the existing `/test` endpoint:

**POST `/explain` endpoint** (response_model=ExplanationResponse):
- Accepts `ExplanationRequest` body and `db: Session = Depends(get_db)`
- Validates ANTHROPIC_API_KEY exists (400 if not)
- Queries Question by question_id (404 if not found)
- Builds a system prompt based on `teaching_mode`:
  - If `teaching_mode == "explanation"`: System prompt = "You are a medical education expert. Provide a detailed explanation of this USMLE question. Explain why the correct answer is right and why each incorrect option is wrong. Include relevant pathophysiology, clinical reasoning, and high-yield facts for exam preparation. Be thorough but concise."
  - If `teaching_mode == "socratic"`: System prompt = "You are a Socratic medical tutor. Instead of giving the answer directly, guide the student through clinical reasoning with targeted questions. Ask one question at a time to help them arrive at the correct understanding. Be encouraging but rigorous. If the student has answered incorrectly, help them understand why their reasoning was flawed without directly stating the answer."
  - Returns 400 if teaching_mode is not "explanation" or "socratic"
- Builds the user message:
  - Includes the question stem, all answer options (label + text), the correct answer label
  - If `user_answer_label` is provided, includes "The student selected: {label}"
- If `conversation_history` is non-empty, appends it to the messages array (for Socratic follow-up turns)
- Calls `client.messages.create(model="claude-sonnet-4-20250514", max_tokens=1500, system=system_prompt, messages=messages_list)`
- Wraps in try/except, raises HTTPException 502 on failure
- Returns ExplanationResponse with content, model, teaching_mode, question_id
