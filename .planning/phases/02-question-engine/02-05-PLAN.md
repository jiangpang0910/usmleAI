"use client" component. JSDoc on every function and interface.

Props:
```typescript
interface QuestionGridProps {
  totalQuestions: number;          // Total number of questions in the session
  currentIndex: number;            // Currently active question (0-indexed)
  questionStates: Map<number, QuestionState>; // Per-question state from QuestionView
  onJumpTo: (index: number) => void; // Callback to navigate to a specific question
  isOpen: boolean;                 // Whether the grid drawer is open
  onToggle: () => void;            // Toggle drawer open/close
}
```

Implementation:
- Render as a drawer/sidebar panel that slides in from the right on toggle
- Use fixed positioning: `fixed right-0 top-0 h-full w-72 bg-white shadow-lg z-50 transform transition-transform` with `translate-x-full` when closed, `translate-x-0` when open
- Include a semi-transparent backdrop overlay when open (click to close)
- Header: "Question Navigator" title with close button (X)
- Grid of numbered squares: `grid grid-cols-5 gap-2 p-4`
- Each square shows the question number (1-indexed)
- Color coding:
  - Unanswered: `bg-gray-100 border border-gray-300 text-gray-700` (default)
  - Answered correctly: `bg-green-100 border border-green-400 text-green-800`
  - Answered incorrectly: `bg-red-100 border border-red-400 text-red-800`
  - Flagged (unanswered): `bg-yellow-100 border border-yellow-400 text-yellow-800` with a small flag icon
  - Flagged (answered): the answered color with a small flag icon overlay in the corner
  - Current question: thick blue border ring `ring-2 ring-blue-600`
- Each square is clickable: `onClick={() => onJumpTo(index)}`
- Below the grid: summary stats line: "Answered: X/{total} | Flagged: Y"
- At bottom: "Review Flagged" button (variant="outline") that jumps to the first flagged question
