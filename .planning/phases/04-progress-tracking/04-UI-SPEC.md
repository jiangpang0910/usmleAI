## Interaction Contracts

### Time Range Toggle (Score History)
- 4 buttons in a row: "7 days", "30 days", "90 days", "All time"
- Active state: primary bg with primary-foreground text
- Inactive state: muted bg with muted-foreground text
- On click: refetch chart data for selected range, update chart with fade transition
- Default selection: "30 days"

### Topic Drill-Down (Score History)
- Click a topic name in the chart legend to toggle its line visibility
- Click "Overall" label to return to aggregate view
- Chart tooltip on hover: shows date, accuracy %, questions answered that day

### Weak Areas List
- Sorted by accuracy ascending (weakest first)
- Each row: rank number, topic name, accuracy %, trend arrow (colored)
- Trend arrows: up arrow (success green) = improving, right arrow (warning amber) = flat, down arrow (destructive red) = declining
- "Practice Weak Areas" button at bottom: navigates to adaptive session with weakest 3 topics pre-selected
- If fewer than 3 topics have data, show all available

### Due for Review
- Card displays: emoji icon, "Due for Review" heading, "{count} questions due for review" body text
- "Start Review" button: navigates to a quiz session with only due-for-review questions
- When count is 0: card is hidden entirely (not shown with "0 questions")

### Topic Coverage Grid
- One row per topic the user has ever practiced
- Progress bar: filled portion uses primary color, unfilled uses muted
- Percentage label right-aligned
- Coverage = (unique questions answered in topic) / (total questions available in topic) * 100
- Sorted alphabetically by topic name

### KPI Cards
- Each card: emoji icon top-left, label below icon, display value (32px semibold) centered, optional small delta text below value
- Delta text: "+5% this week" in success green, or "-3% this week" in destructive red
- No delta shown if insufficient historical data

## Chart Specification

### Score History Line Chart (Recharts)
- Type: LineChart with CartesianGrid, XAxis, YAxis, Tooltip, Legend
- X-axis: date labels (format: "Mar 15" for 7d/30d, "Mar 2026" for 90d/all)
- Y-axis: 0-100% accuracy scale
- Line: stroke color = primary (hsl(217 71% 25%)), strokeWidth = 2, dot radius = 4
- Grid: dashed lines using border color
- Tooltip: white bg card with shadow, shows date + accuracy + question count
- Responsive: use ResponsiveContainer with 100% width, 300px height (250px on mobile)
- Animation: 300ms ease-in-out on data change

### KPI Metric Display
- No chart component; pure text layout inside Card
- Display value: 32px semibold, foreground color
- Label: 12px semibold, muted-foreground color
- Delta: 12px regular, success/destructive color based on direction

### Topic Coverage Progress Bars
- Height: 8px with rounded ends (border-radius sm)
- Fill: primary color for completed portion
- Track: muted color for remaining portion
- Label: topic name left, percentage right, body size (14px)
