       - `get_redis_client() -> redis.Redis`: Returns a Redis client using settings.REDIS_URL. Use `redis.from_url()` with `decode_responses=True`. Handle connection errors gracefully (log warning, return None).
       - `build_cache_key(prefix: str, **params) -> str`: Builds a deterministic cache key from prefix and sorted query params. Example: "topics:discipline=basic_science&step=step1". Skip None-valued params.
       - `cache_get(key: str) -> Optional[str]`: Get value from Redis. Returns None on miss or connection error.
       - `cache_set(key: str, value: str, ttl: int = 300) -> None`: Set value in Redis with TTL (default 5 minutes). Silently handles connection errors.
       - `cache_delete_pattern(pattern: str) -> None`: Delete all keys matching a glob pattern (e.g., "topics:*"). For cache invalidation.

    All functions must have docstring comments explaining their purpose. Use `import json` for serialization (done in routers, not in cache module — cache module works with raw strings).

    Install the new dependency: run `pip install redis==5.2.1` in the backend venv.
