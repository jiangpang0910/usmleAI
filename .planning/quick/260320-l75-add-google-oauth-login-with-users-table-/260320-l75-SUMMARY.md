## Decisions Made
- Used authlib with Starlette integration for OAuth (supports OpenID Connect discovery, well-maintained)
- python-jose for JWT (lightweight, widely used with FastAPI)
- Stateless JWT with 24h expiry -- logout is client-side token deletion
- Token passed via URL query param from OAuth callback (simple, no cookie complexity)
- Created AuthProviderWrapper as "use client" component since root layout is a server component

## User Setup Required

**External services require manual configuration.** To complete Google OAuth setup:

1. Go to [Google Cloud Console](https://console.cloud.google.com/) -> APIs & Services -> Credentials
2. Create an OAuth 2.0 Client ID (Web application type)
3. Add `http://localhost:8000/api/auth/google/callback` to Authorized redirect URIs
4. Add `http://localhost:3000` to Authorized JavaScript origins
5. Copy the Client ID and Client Secret to `backend/.env`:
   ```
   GOOGLE_CLIENT_ID=your-client-id-here
   GOOGLE_CLIENT_SECRET=your-client-secret-here
   ```
6. Optionally generate a secure JWT secret: `python3 -c "import secrets; print(secrets.token_hex(32))"`
